2. Конкретные промпты и полученная помощь
Промпт 1:
"Ошибка ENOENT при запуске npm install в пустой папке. Как правильно создать проект через Vite?"

Результат: ИИ объяснил, что npm install требует наличия package.json. Получил правильную команду: npm create vite@latest . -- --template react. Это помогло корректно инициализировать структуру проекта для Lab 1.2.

Промпт 2:
"Объясни разницу между императивным подходом jQuery и декларативным в React на примере переключения CSS класса."

Результат: Получил концептуальное объяснение для Task 3 (Lab 1.1). Понял, что в jQuery я вручную манипулирую DOM (imperative), а в React я просто меняю состояние (state), и интерфейс обновляется автоматически (declarative).

Промпт 3:
"Как работает Virtual DOM в React при обновлении счетчика?"

Результат: ИИ объяснил процессы diffing (сравнение) и patching (применение изменений). Это помогло мне написать теоретическую часть для Lab 1.2 о том, почему React обновляет только текст цифры, а не всю страницу.

Промпт 4:
"Почему в React нельзя использовать document.querySelector для изменения текста?"

Результат: Получил разъяснение, что React работает как "View Layer" и управляет DOM самостоятельно. Использование прямых манипуляций нарушает декларативную логику и может привести к багам.
