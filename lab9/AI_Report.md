## 1) Проблема
Проект не собирался и не запускался корректно из-за ошибок TypeScript в компоненте виртуализированного списка.

### Зафиксированные ошибки
- `TS2305`: в `react-window` нет экспорта `FixedSizeList`.
- `TS7031`: неявный тип `any` у `index` и `style`.

Дополнительно было замечено, что запуск команд из корня `frontend` давал `ENOENT` (не найден `package.json`), поэтому команды нужно выполнять из папки `lab9/task1/perf-app`.

## 2) Причина
В проекте установлена библиотека `react-window@2.x`, а код был написан под API версии `1.x`:
- использовался `FixedSizeList` (старый API),
- рендер-колбэк в children вместо `rowComponent` + `rowProps`.

## 3) Что изменено
Изменён файл:
- `lab9/task1/perf-app/src/components/VirtualList.tsx`

Сделано:
- импорт заменён на `List` и `RowComponentProps` из `react-window`;
- добавлен типизированный компонент строки `Row`;
- данные передаются через `rowProps`;
- список переведён на API v2: `rowComponent`, `rowCount`, `rowHeight`, `rowProps`, `style`;
- устранены ошибки типизации TypeScript.
