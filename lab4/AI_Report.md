4. Что я узнал (Key Learnings)
Synthetic Events: Понял, что e.preventDefault() критически важен в React для предотвращения стандартного поведения браузера при отправке форм.
Controlled vs Uncontrolled: Научился полностью контролировать ввод пользователя через value и useState.
Refactoring: Осознал, как разбиение кода на мелкие части (ArticleItem, AddArticle) делает проект читаемым и легким в тестировании.
Lifting State Up: Попрактиковался в передаче функций-обработчиков вниз по дереву компонентов через пропсы.
Project Setup: Освоил профессиональную настройку окружения через Vite.
